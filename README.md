# Instructions

[Reference Instructions](https://pkgdocs.julialang.org/dev/apps/)

A. Create `MyReverseApp` GitHub repository

1. Create a Julia App: folder MyReverseApp : pkg> generate MyReversApp: src/MyReverseApp, Project.toml
2. Install a Julia App: julia> app add MyReverseApp
3. Run the Julia app `reverse` with args to reverse: $ reverse one two three
